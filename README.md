# java-racingcar-precourse

> Evnet-Driven-Architecture로 구현한 Java 자동차 경주

## 도메인

### 🧩 Common
- [x] Event Bus 
  - [x] 전역 스레드에서 발생하는 Excption 처리 
- [x] Event Bus 에 사용될 이벤트 정의
### ⚙️ Config
- [x] 설정 클래스

### 🏎️ Entry
- [x] 입력받은 참가자 문자열 파싱한다.
- [x] 입력받은 참가자 문자열을 검증한다.
- [x] 자동차를 움직인다.
- [x] 자동차의 움직인 거리를 반환한다.

### 🛣️ Lap
- [x] 랩 횟수가 남았다는 이벤트를 받으면 랩 진행중 상태로 바꾸고 랩 진행 이벤트 반환
- [x] 경기 기록됨 이벤트를 받으면 랩 대기중 상태로 바꾸고 랩 종료 이벤트를 반환

###  💻 I/O
- [x] 사용자로부터 참여할 자동차 목록을 입력 받는다.
- [x] 사용자로부터 시도할 횟수(Lap Count)를 입력 받는다.
- [x] 실행 결과를 출력한다.

### 🏟️ RACE
- [x] 남은 랩 횟수에 따른 이벤트를 발행한다.
  - [x] 남은 랩 횟수가 없으면 경기 종료 이벤트를 발행한다.
  - [x] 남은 랩 횟수가 있으면 랩 횟수 남음 이벤트를 발행한다.

### 📝 SCORE
- [x] 경주 현황을 기록한다.
- [x] 경주 승리자를 기록한다.
- [x] 경주 결과를 반환한다.
