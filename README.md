# java-racingcar-precourse

> 자동차 경주
 
## 기능 목록

### 1. 입력 출력

- [ ] 안내 메시지를 출력한다.
- [ ] 사용자 입력을 받는다.
- [ ] 경기 현황을 출력한다.

### 2. 파싱

- [ ] 입력 받은 자동차 이름 문자열을 경주용 자동차로 변환한다.
- [ ] 시도할 횟수를 숫자로 변환한다.

### 3. 경주

#### 패독
- [ ] 문자열(자동차 이름 목록)을 받고 자동차 목록을 반환한다.
- [ ] 문자열(자동차 이름)을 받고 자동차 생성한다.

#### 자동차
- [ ] 움직인다.
- [ ] 움직인 거리를 반환한다.

#### 경기장
- [ ] 자동차를 움직인다.
- [ ] 자동차 현재 정보를 반환한다.
